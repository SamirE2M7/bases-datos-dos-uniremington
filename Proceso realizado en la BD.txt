Aqui en este espacio voy a detallar paso a paso para la elaboracion de mi BD:

1. El problema planteado es el sgte: "una empresa de ventas en línea que vende una variedad de productos.
Quieres diseñar una base de datos para gestionar los pedidos de los clientes, los productos disponibles y 
la información de los clientes.
2. Practicar sentencias SQl e implementar procedimiento almacenados". 
3. Diseño de la base de datos se llama: ventas
4. Diseño de las tablas: Clientes, detalles_pedidos, pedidos y productos.
5. Detalles de las tablas:

-- Tabla de Clientes
CREATE TABLE clientes (
    ID_Cliente INT PRIMARY KEY,
    Nombre VARCHAR(100),
    Direccion VARCHAR(255),
    CorreoElectronico VARCHAR(100)
);

-- Tabla de Productos
CREATE TABLE productos (
    ID_Producto INT PRIMARY KEY,
    Nombre VARCHAR(100),
    Precio DECIMAL(10, 2),
    Stock INT
);

-- Tabla de Pedidos
CREATE TABLE pedidos (
    ID_Pedido INT PRIMARY KEY,
    ID_Cliente INT,
    Fecha DATE,
    Total DECIMAL(10, 2),
    FOREIGN KEY (ID_Cliente) REFERENCES Clientes(ID_Cliente)
);

-- Tabla de Detalles del Pedido
CREATE TABLE detalles_pedido (
    ID_Detalle INT PRIMARY KEY,
    ID_Pedido INT,
    ID_Producto INT,
    Cantidad INT,
    Precio_Unitario DECIMAL(10, 2),
    Subtotal DECIMAL(10, 2),
    FOREIGN KEY (ID_Pedido) REFERENCES Pedidos(ID_Pedido),
    FOREIGN KEY (ID_Producto) REFERENCES Productos(ID_Producto)
);  

6. A cada tabla se le hizo minimo 10 registros.
7. Para hacer sentencias SQL: 
-> Para consultar una tabla, por ejemplo pedidos, usamos el sgte codigo: select * from pedidos; Me muestra la tabla pedidos.
-> Para una consulta mas especifica de una tabla, el sgte codigo: select Fecha from pedidos; Me muestra una columna dentro de pedidos.
-> Para aun mas especifica con detalles dentro de una columna, el sgte codigo: select * from productos where Stock = 50; Me muestra los productos que tengan solo Stock de 50.
-> Para contar cuantos clientes tengo en la tabla cliente, el sgte codigo: select count(ID_Cliente) from clientes. Me muestra el total de clientes registrados.
-> En el caso de un registro nuevo, por ejemplo en clientes: Usamos el sgte codigo: insert into clientes values (null, 'nombre','direccion','email');
